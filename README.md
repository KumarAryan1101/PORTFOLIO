# Kumar Aryan — Portfolio 🚀

My personal portfolio — an interactive, motion-first website showcasing my projects, product thinking and campus leadership.

**Author:** [Kumar Aryan](https://www.linkedin.com/in/kumar-aryan-871294333) · ECE @ IIIT Nagpur '28

## ✨ Highlights

- **WebGL flashlight-reveal hero** — custom GLSL shader (Bayer dithering + wave distortion) that reveals the image in color under your cursor
- **Ambient frame-sequence animation** — 45-frame canvas loop driven by GSAP's ticker, DPR-aware and paused when offscreen
- **Neo-brutalist design system** — thick borders, hard offset shadows, 3D-tilt cards, marquees and spring physics everywhere
- **Fully responsive** — tuned for phones, tablets and desktops (capped pixel ratios, low-power GPU hints, single-column collapse)
- **5 pages** — Home, Projects, Product Thinking, About, Contact with animated route transitions

## 🛠️ Tech Stack

| Layer | Tech |
|---|---|
| Framework | React + TypeScript + Vite |
| Styling | Tailwind CSS |
| Animation | Framer Motion, GSAP |
| 3D / Shaders | Three.js, @react-three/fiber, @react-three/drei, custom GLSL |
| Routing | React Router |

## 🏃 Run locally

```bash
npm install
npm run dev      # dev server at http://localhost:5173
npm run build    # production build in dist/
```
